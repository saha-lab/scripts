The scripts in this folder is meant for quality analysis after an alignment is done (usually with hisat2). They are run on a directory of bam files.

————————————————
iterate_cleanbam
————————————————
A script used in the ‘RNA-Seq Alignment Quality Analysis’ protocol as a pre-processing step to clean alignments from a bam file in order to run BAMQC on it.

——————————————
rseqc_automate
——————————————
A script used in the ‘RNA-Seq Alignment Quality Analysis’ protocol to run RSeQC modules on a directory of bam files.
